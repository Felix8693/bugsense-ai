# BugSense AI Roadmap

BugSense AI is an open-source AI debugging assistant for developers and AI coding tool users.  
This roadmap describes the planned direction of the project.

## v0.1.0 - Initial MVP

- FastAPI backend
- Next.js frontend
- Redis cache support
- Docker Compose setup
- Basic error log analysis workflow
- Online demo deployment
- Support for common error types:
  - Python errors
  - Node.js errors
  - Docker errors
  - Next.js errors
  - Git errors
  - CI/CD errors
  - AI coding tool errors

## v0.2.0 - Better Debugging Experience

- Add more real-world error examples
- Improve error classification
- Improve root cause analysis
- Add regression tests for the log parser
- Improve Windows setup documentation
- Add clearer frontend result cards
- Improve Docker and deployment troubleshooting docs

## v0.3.0 - Open Source Maintainer Tools

- GitHub issue triage assistant
- CI/CD failure analysis assistant
- Pull request debugging helper
- Automatic test suggestion for bug reports
- Better contributor onboarding documentation
- More examples for open-source maintainers

## v0.4.0 - Multi-Provider AI Support

- Support OpenAI provider
- Support Anthropic provider
- Support MiMo provider
- Add provider switching in configuration
- Improve fallback behavior when one provider fails

## Long-Term Goals

- Help new open-source contributors understand confusing errors faster
- Reduce repetitive debugging support work for maintainers
- Make AI-powered debugging easier to self-host
- Build a practical developer tool for logs, build failures, and AI coding tool errors
